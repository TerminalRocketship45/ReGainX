# ReGainX

ReGainX is a project focused on developing exoskeleton technologies for assisting individuals with Parkinson's disease. This repository contains simulations, training code, data, and results for elbow and leg exoskeletons.

## Structure

- `docs/`: Documentation, website files, and Jupyter notebooks for simulations and proofs.
- `exoskeletons/`: Code and data for elbow and leg exoskeleton projects.
  - `elbow/`: Elbow exoskeleton development.
  - `leg/`: Leg exoskeleton development.
- `media/`: Media files, videos, plots, and project gallery.
